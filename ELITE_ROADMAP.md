# Elite 应用化路线（行业顶尖目标）

## 阶段 A：稳定测色（已完成基础）

- 单图现场模式 + 双图标准模式
- 自动材质档位 + 置信度
- 自动输出报告与可视化工件
- 手动ROI/四点兜底 + 批量目录跑批
- 阈值覆盖参数（按产线灵活收紧/放宽）
- ArUco 工位定位（高一致性拍摄场景）
- HTML 诊断页（生产/品质可直接查看）
- 多张融合判定（同批次稳定性增强）
- 清晰度/光照质量旗标 + 自动重拍指导

## 阶段 B：产线闭环

- FastAPI 服务层（single/dual/batch/ensemble/history）已落地，可直接对接 MES/QMS
- 历史质量数据库 + 漂移评估 + 查询接口已落地（overview/runs）
- 工艺建议闭环已落地：规则引擎自动从 dL/dC/dH 输出调参动作与风险等级
- 决策中心已落地：自动给出放行策略 + 客户/老板/公司三视角评分 + 成本预估
- 经营接口已落地：`history/executive` 直接输出管理层指标
- 闭环学习已落地：支持客诉/退货结果回写与策略调优建议（policy recommendation）
- 策略实验室已落地：Policy Lab 数字孪生仿真 + Pareto 多目标选优
- 反事实孪生已落地：Counterfactual Twin（Conformal 区间 + 三类策略推荐）
- 客户分层策略已落地：支持 `vip/standard/growth/economy` 自动套用策略补丁
- 客诉早预警已落地：`history/early-warning` 输出风险等级、驱动因子、7/30 天预测
- Champion-Challenger 已落地：自动给出 `PROMOTE/CANARY/REJECT` 与灰度计划
- 开放算法编排已落地：LinUCB 上下文赌博机策略推荐（`history/open-bandit-policy`）
- v14 创新引擎已落地：光谱重建同色异谱 + 纹理感知色差 + 老化预测 + 自动墨量处方
- v14 运营创新已落地：批次混拼优化 + 客户容忍度学习 + 数字色彩护照
- 接入相机 SDK（固定曝光/白平衡锁定）
- 加入日常标定任务（灰卡 + ColorChecker）
- 规则建议 -> 工艺参数映射（墨量/线速/温区）

## 阶段 C：全场景泛化（下一阶段）

- 多工厂域自适应（每线阈值自学习）
- 反光场景偏振补偿
- 透明膜背板标准化（黑白双底板建模）

## 阶段 D：智能调参

- 训练 `参数 -> 色差` 回归模型
- 采用小步闭环（单轮参数改动上限）
- 引入异常检测与回滚策略

## 上线门槛（建议）

- 连续 500 批误判率 < 1%
- 低置信度样本自动转人工复核
- 关键工序全链路可追溯（图像、参数、结论）
